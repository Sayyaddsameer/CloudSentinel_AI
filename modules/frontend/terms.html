Description of Service

CloudSentinel AI is a cloud security intelligence platform that:

  • Performs read-only scans of your AWS and/or GCP cloud environments
  • Detects security misconfigurations, IAM policy issues, and compliance gaps
  • Provides AI-generated remediation recommendations
  • Stores scan results (risk records) in AWS DynamoDB within our infrastructure

Read-Only Access Guarantee

CloudSentinel operates with a strict read-only access model:

  • We deploy an IAM role in your AWS account via CloudFormation with read-only permissions only
  • We will never create, modify, or delete any resource in your account
  • GCP service account keys are stored encrypted in AWS Secrets Manager and never logged
  • You can revoke all access instantly by deleting the CloudSentinel-Scanner CloudFormation stack

Data Retention

We do not store the contents of your cloud resources — only metadata about detected risk conditions (resource names, risk types, and remediation descriptions).
